Rust isn't supported on managed Nubo yet.

Nubo builds your app with Cloud Native Buildpacks, and the current builder doesn't include a Rust buildpack. There also isn't a Dockerfile or pre-built-image option yet, so there's no way to ship a Rust binary on the managed platform today.

We're working on wider build support. In the meantime:

- Reach out at `support@withnubo.com` or on [Discord](https://discord.com/invite/4SM4faYqWQ) and we'll tell you where Rust support stands.
- For anything you deploy on Nubo right now, use one of the [supported languages](/deploy-guides/golang).

We'll replace this page with a real walkthrough as soon as Rust builds land.
